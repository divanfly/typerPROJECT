# Typer - Тренажёр слепой печати

![Java](https://img.shields.io/badge/Java-21-orange)
![JavaFX](https://img.shields.io/badge/JavaFX-21-blue)
![Maven](https://img.shields.io/badge/Maven-3.9.6-red)

Десктопное приложение для тренировки скорости и точности набора текста, вдохновлённое популярным веб-сервисом [MonkeyType](https://monkeytype.com/).

## 📸 Скриншоты

```
┌─────────────────────────────────────────┐
│            [EN] [RU]                    │
├─────────────────────────────────────────┤
│              typer                      │
├─────────────────────────────────────────┤
│                                         │
│   the would can bec|ause give our us    │
│                                         │
├─────────────────────────────────────────┤
│         WPM        accuracy             │
│          42          95%                │
│        time: 12.5s                      │
│     press F1 to restart                 │
└─────────────────────────────────────────┘
```

## ✨ Возможности

- 🌍 **Два языка** — английский и русский
- 🎯 **10 случайных слов** — каждый тест уникален
- ⌨️ **Живой курсор** — палочка `|` показывает текущую позицию
- 🔴 **Подсветка ошибок** — красный цвет для неправильных букв
- 📊 **Статистика WPM** — слова в минуту
- 📈 **Точность (Accuracy)** — процент правильных символов
- 🔄 **F1 перезапуск** — новый тест в любой момент
- ⚡ **Автозавершение** — не нужен пробел после последнего слова

## 📦 Установка и запуск

### Требования

- Java JDK 17 или выше
- Maven 3.9+ (или будет скачан автоматически)

### Способ 1: Через Maven

```powershell
cd typer
mvn javafx:run
```

### Способ 2: Через batch-файл (Windows)

```powershell
.\run.bat
```

### Способ 3: Сборка JAR

```powershell
cd typer
mvn clean package
java -jar target/typer-1.0-SNAPSHOT.jar
```

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `A-Z`, `А-Я` | Ввод символов |
| `Пробел` | Переход к следующему слову |
| `Backspace` | Удаление последнего символа |
| `F1` | Перезапуск теста (новые слова) |

## 🧠 Алгоритмы

### Расчёт WPM (Words Per Minute)

```
WPM = (Правильные символы ÷ 5) ÷ Время в минутах
```

> Деление на 5, потому что средняя длина слова ≈ 5 символов (стандарт)

### Расчёт точности (Accuracy)

```
Accuracy = (Правильные символы ÷ Всего символов) × 100%
```

## 📝 Архитектура

### Основной класс `TyperApp`

```java
public class TyperApp extends Application {
    
    // Константы цветов и словари
    private static final String[] ENGLISH_WORDS = {...};  // 100 слов
    private static final String[] RUSSIAN_WORDS = {...};  // 100 слов
    
    // Состояние приложения
    private List<String> words;           // Текущие 10 слов
    private int currentWordIndex;         // Индекс слова
    private StringBuilder currentInput;   // Ввод пользователя
    private long startTime;               // Время старта
    private boolean isRussian;            // Флаг языка
    
    // Ключевые методы
    void start(Stage stage);              // Инициализация UI
    void generateWords();                 // Генерация теста
    void updateWordsDisplay();            // Обновление отображения
    void processWordComplete();           // Обработка слова
    void finishTest();                    // Завершение и статистика
    void restartTest();                   // Перезапуск
}
```

### Диаграмма состояний

```
┌─────────────┐     F1      ┌─────────────┐
│   ГОТОВ     │ ──────────> │   ГОТОВ     │

└──────┬──────┘             └─────────────┘
       │ первая буква              ↑

┌─────────────┐                    │

       ▼                           │
┌─────────────┐                    │

└─────────────┘
```
